# IsoChrome (WPF)

**IsoChrome** — портативное WPF-приложение для управления изолированными сессиями Google Chrome: быстрые временные сессии, постоянные профили, закладки, история запусков.

> Текущая версия: **0.4.71**

## Возможности
- **Быстрая сессия**
  - запуск отдельного экземпляра Chrome с временным профилем;
  - инкогнито и User-Agent (по умолчанию/случайный/свой);
  - список активных сессий, открытие URL в выбранной, закрытие выбранной/всех.
- **Профили**
  - создание и запуск постоянных профилей;
  - режимы профиля: инкогнито, отключение фонового режима, пользовательский User-Agent;
  - переименование (с переименованием папки), очистка данных, удаление.
- **Закладки**
  - хранилище `bookmarks.json` + синхронизация в `bookmarks.txt` для совместимости;
  - редактирование: название, URL, группа, теги (чипсы), заметка, закрепление;
  - импорт/экспорт TXT/CSV/JSON;
  - фильтр по группе и поиск по всем полям.
- **Недавние**
  - журнал запусков и быстрый повторный запуск.

## Установка и запуск
- Скачайте последнюю версию (portable) из раздела релизов репозитория.
- Распакуйте архив в любую папку.
- Запустите `IsoChrome.exe`.

## Хранение данных
По умолчанию данные хранятся в профиле пользователя Windows:
- `%AppData%\IsoChrome\profiles\` — постоянные профили.
- `%AppData%\IsoChrome\bookmarks.json` — основное хранилище закладок.
- `%AppData%\IsoChrome\bookmarks.txt` — текстовая копия закладок для совместимости.

Быстрые временные профили создаются отдельно (для стабильности путей) и автоматически очищаются при закрытии сессии.
